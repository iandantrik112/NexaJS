# NexaGeolocation — catatan sumber

Dokumentasi web resmi NexaJS untuk modul ini:

- **URL:** `/docs/platform/javascript/nexa-geolocation`
- **Template konten:** `templates/theme/docs/platform/javascript/nexa-geolocation.html`
- **Menu sidebar:** `templates/theme/docs/platform/javascript/menu.html` (cabang *NexaGeolocation() — GPS*)
- **Implementasi:** `assets/modules/Geolocation/NexaGeolocation.js` (ekspor juga lewat `assets/modules/Nexa.js`)
